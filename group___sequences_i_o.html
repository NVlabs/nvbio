Sequence data input
IO

Detailed Description

This module contains a series of classes to load and represent read streams. The idea is that a read stream is an object implementing a simple interface, SequenceDataStream, which allows to stream through a file or other set of reads in batches, which are represented in memory with an object inheriting from SequenceData. There are several kinds of SequenceData containers to keep the reads in the host RAM, or in CUDA device memory. Additionally, the same container can be viewed with different SequenceDataView's, in order to allow reinterpreting the base arrays as arrays of different types, e.g. to perform vector loads or use LDG.
